Adding your stuff to github

Since is something so many ask for, belowis a screencast about how to push something onto github and here is what you should type into your terminal

  • Navigate to your github repository from top level folder: cd ./desktop/repositories/____.github.io
  • Add your changes:
    git add .
  • Commit your changes: git commit -m "write a message to yourself"
  • push your changes online: git push
